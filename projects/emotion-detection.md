# Emotion Detection Using Multi-Signals

Timeline: July 2024 – September 2024

Overview

A multi-modal emotion recognition system that fuses text, audio, and facial-expression streams using specialized encoders and an LSTM-based sequence fusion network. Designed for real-time inference and interactive analysis via Streamlit.

Key Features

- Multi-modal preprocessing pipelines (text tokenization, audio feature extraction, face detection + alignment)
- Encoders for each modality and LSTM fusion for temporal modeling
- Streamlit app for live inference, visualization, and confidence outputs
