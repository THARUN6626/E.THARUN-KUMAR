# Cloud-Based Online Blood Bank System

Timeline: October 2024 – December 2024

Overview

A role-based cloud application for real-time blood inventory management, donor registration, and hospital coordination. Deployed on Microsoft Azure with RBAC, REST APIs, and scalable storage.
