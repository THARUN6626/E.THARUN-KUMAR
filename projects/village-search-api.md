# Village Search API

Timeline: April 2026

Overview

A full-stack geospatial search application featuring a FastAPI backend and PostgreSQL database. Provides efficient village lookup and advanced filtering with a responsive frontend.

Repository

- https://github.com/THARUN6626/Village-api-project

Key Features

- FastAPI-based high-performance API
- PostgreSQL for data persistence and geospatial queries
- Simple HTML frontend for demo and manual testing

Installation & Run

1. Clone: `git clone https://github.com/THARUN6626/Village-api-project`
2. `python -m venv venv && source venv/bin/activate`
3. `pip install -r requirements.txt`
4. `uvicorn app.main:app --reload`

Technologies

FastAPI, PostgreSQL, Python, HTML
